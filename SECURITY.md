# Security Policy

## Supported Versions

The project is currently pre-1.0. Security fixes target the latest `main` branch unless otherwise stated.

## Reporting A Vulnerability

Please report security issues privately through the repository owner before publishing details.

## Local Safety Model

This MCP can create files, launch Cocos Creator, install a project-local editor extension, and run local build commands. Only configure it in trusted developer environments and only grant it access to project directories you are comfortable automating.

This MCP intentionally does not upload builds, submit releases, or manage production app credentials.
