# Changelog

All notable changes to this project will be documented in this file.

## 0.1.0

- Initial local Cocos Creator MCP server.
- Added project creation, scene template creation, editor bridge installation, scene blueprint application, and local WeChat Mini Game build tools.
- Added high-level sprite placement tools for generated assets: `cocos_local_assign_sprite_frame`, `cocos_local_create_sprite_node`, and `cocos_local_place_sprite_assets`.
- Added English and Chinese documentation.
